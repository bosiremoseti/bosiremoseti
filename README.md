# Simple Portfolio Website — Innocent Bosire Moseti

A clean, modern, self-contained single-file portfolio website built from your resume.

## Quick Start

1. Open `index.html` directly in any browser (double-click it).
2. That's it — no build step, no server required.

## What's Included

- Professional hero with headline and contact info
- Profile summary (About)
- Full Employment History (4 roles with key achievements)
- Selected projects derived from real work (colocation, security policy, network operations, infrastructure)
- Skills grouped by area (Networks, Security, Infrastructure)
- Selected Certifications
- Education
- Contact with email, phone, and LinkedIn

## How to Further Customize

- Update the "Resume" download button link (currently `#`) to your actual PDF or Google Drive link
- Add GitHub or other project links in the projects JS data
- Tweak any wording or add more projects in the `const projects = [...]` array near the bottom of index.html
- Add/remove certifications in the Certifications section

## Theme
- Light / dark mode toggle (auto + persists)
- Fully responsive, modern design using Tailwind via CDN

## Deploy
Upload the folder (or just index.html) to GitHub Pages, Netlify, Vercel, or any static host.
